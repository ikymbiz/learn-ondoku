# Obsidian Terminal (Vision Reader & Assessment)

## 概要

Obsidian Terminal は、画像からのテキスト抽出（OCR）、AIによる高品質なテキスト読み上げ（TTS）、およびユーザーの音読に対する高精度な音声認識・採点をワンストップで提供するWebアプリケーションです。

Gemini APIを利用した強力な画像認識と、OpenAI API（TTS / Whisper）を活用した音声処理を組み合わせることで、語学学習、ドキュメントのデジタル化、発音のセルフチェックなど、多様なユースケースに対応します。モダンで洗練されたUIを持ち、設定や履歴はすべてブラウザのローカルストレージに安全に保存されます。

## 主な機能

### 1. Vision Reader (テキスト抽出・読み上げ)
*   **画像からのテキスト抽出**: カメラでの直接撮影、または画像ファイルのアップロードから、Gemini AIモデル（デフォルト: `gemini-2.5-flash`）を使用して高精度にテキストを抽出します。
*   **抽出履歴の管理**: 抽出されたテキストは自動的にローカルデータベースに保存され、後からドロップダウンで簡単に呼び出すことができます。
*   **AI音声読み上げ (TTS)**: 抽出したテキストを、OpenAIのText-to-Speech APIを使用して自然な音声で読み上げます。再生・一時停止・停止のコントロールが可能です。

### 2. Reading Assessment (音読採点)
*   **シームレスな連携**: Vision Readerで抽出したテキストをワンクリックで採点画面にインポートできます。
*   **高精度な音声認識**: デバイスのマイクから録音したユーザーの音声を、OpenAI Whisper APIを利用して正確に文字起こしします。
*   **自動スコアリング**: ターゲットテキストと実際に読み上げられたテキストを比較・解析し、「発音の正確さ」や「読み上げるスピード」を自動で採点・視覚化します。
*   **カスタム採点基準**: ユーザー自身で独自の採点基準（例: 表現力、声の大きさなど）を追加・管理することが可能です。

### 3. パーソナライズ設定
*   **APIキーの安全な保存**: Gemini APIとOpenAI APIのキーは、IndexedDBを用いてブラウザ内にセキュアに保存されます。
*   **柔軟な音声カスタマイズ**: 言語の自動検出/手動設定、6種類のOpenAIボイスモデルの選択、話速（0.25〜4.0倍）や音量の細かい調整が可能です。

## 使用技術・ライブラリ

*   **フロントエンド**: HTML5, CSS3, Vanilla JavaScript (フレームワーク非依存の軽量構成)
*   **デザイン・タイポグラフィ**: CSS Variablesによるカスタムデザインシステム, Google Fonts (Syne, DM Sans, JetBrains Mono)
*   **アイコン**: Phosphor Icons
*   **主要Web API**:
    *   `MediaDevices API` (カメラおよびマイクのストリーム取得)
    *   `MediaRecorder API` (音声の録音処理)
    *   `IndexedDB` (設定およびOCR履歴の永続化)
    *   `Wake Lock API` (処理中や録音中のスリープ防止)
*   **外部AIサービス (API連携)**:
    *   Google Gemini API (VisionOCR用)
    *   OpenAI API (TTS用, Whisper音声認識用)

## セットアップ・使い方

### 必要な環境
*   マイクやカメラにアクセスするため、**ローカル環境（`http://localhost`）** または **HTTPS通信** が有効なサーバー上で実行してください。
*   モダンなWebブラウザ（Google Chrome, Microsoft Edge, Safariの最新版）を推奨します。

### インストール手順
1.  本リポジトリのHTMLファイルをダウンロード、またはクローンします。
2.  VS Codeの Live Server 拡張機能などを使用して、ローカルサーバーを立ち上げます。
3.  ブラウザで該当のURL（例: `http://localhost:5500`）にアクセスします。

### 初期設定（必須）
アプリを正常に動作させるために、各AIサービスのAPIキーを設定する必要があります。

1.  画面右上の **「設定 (歯車アイコン)」** タブを開きます。
2.  以下のAPIキーを取得し、入力します。
    *   **Gemini API Key**: Google AI Studioから取得（Vision機能に使用）
    *   **OpenAI API Key**: OpenAI Developer Platformから取得（音声合成・音声認識に使用）
3.  必要に応じて音声設定（ボイスの種類や話速など）を調整し、**「設定を保存」** ボタンをクリックします。

### 基本的な使い方

**画像を読み込んでテキストを抽出する**
1.  **「Vision Reader (カメラアイコン)」** タブを開きます。
2.  「カメラ」ボタンで撮影するか、「ファイル」ボタンでデバイス内の画像を選択します。
3.  画像が表示されたら「テキストを抽出」ボタンをクリックします。
4.  抽出完了後、「音読 (OpenAI)」ボタンを押すとテキストが読み上げられます。

**音読のテストと採点を行う**
1.  抽出されたテキストボックスの右上にある「採点タブへ送る（マイクのアイコン）」ボタンをクリックします。
2.  **「Reading Assessment (マイクアイコン)」** タブに自動で移動します。
3.  「録音開始」ボタンを押し、表示されているテキストをマイクに向かって音読します。
4.  読み終わったら「録音停止」ボタンを押します。
5.  AIによる音声認識が行われ、数秒後に画面下部に総合スコアと項目別の採点結果が表示されます。
